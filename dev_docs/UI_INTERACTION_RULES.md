# Interaction Rules

---

# Selection Model

The application uses a global selection system.

Selecting an entity updates:

Inspector Panel
Graph Highlight
Timeline Highlight

Entities:

Character
Event
Location
World Item

---

# Navigation

Navigation must support:

ActivityBar click
Sidebar click
Command Palette
Search result click

---

# Context Menu

Right click on:

Timeline Event
Graph Node
Character

Shows menu:

Edit
Open
Delete
Link

---

# Drag and Drop

Supported in:

Timeline
Chapter ordering
Graph nodes

---

# Cross Navigation

Timeline → Writing Studio

Open Scene button

Timeline → Graph

Highlight node

Character → Timeline

Show character events

---

# Confirm Workflow

Generated characters appear as:

Candidate (grey)

User must click:

Confirm ✓

Confirmed characters are used in story generation.

---

# Undo / Redo

All editing actions must support undo/redo.

Supported:

Character edits
Timeline edits
World edits
Writing edits
