==================================================
ROUTE RULES
==================================================

Rule 1:
Every route must render a workspace layout.

Rule 2:
Routes must never crash or show blank screen.

Rule 3:
Deep link navigation must select the correct entity.

Example:

timeline -> open scene

/timeline/event/123
→
/writing/scene/456

Rule 4:
Navigation must preserve sidebar state when possible.

Rule 5:
Invalid IDs must show:

"Entity not found"

with navigation back option.

TEST REQUIREMENTS

- no console errors
