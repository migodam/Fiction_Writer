# Narrative IDE Component Rules

All UI components must follow consistent patterns.

---

# Buttons

Primary button

Blue

Used for:

Create
Generate
Run

Secondary button

Gray

Used for:

Edit
Open
View

Danger button

Red

Used for:

Delete
Reset

---

# Icon Buttons

Used in:

Toolbar
Timeline
Graph

Must include tooltip.

---

# Lists

Lists must support:

Search
Filter
Sort

Examples:

Character List
Timeline Events
World Items

---

# Cards

Used for:

Characters
World Items
Reader Profiles

Card fields:

Title
Tags
Quick Actions

---

# Editor

Writing editor must support:

Markdown
Rich Text
Inline highlights

Toolbar actions:

Generate Scene
Rewrite
Continue
Improve Dialogue

---

# Graph Components

Nodes:

Character
Event
Location
Object

Edges:

participates
causes
reveals
conflicts

---

# Timeline Nodes

Timeline events appear as nodes.

Node displays:

Title
Time
Participants

Hover shows summary.

---

# Charts

Used in:

Simulation
Insights

Charts:

Line chart
Bar chart
Heatmap
