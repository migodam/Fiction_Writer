# Task Pack Template

Use this template for every parallel task. Do not start implementation until every field is populated.

## Task Pack Schema
- `task_id`
- `title`
- `owner_role`
- `goal`
- `non_goals`
- `owned_paths`
- `forbidden_paths`
- `shared_surfaces`
- `blocked_by`
- `prerequisites`
- `deliverables`
- `definition_of_done`
- `required_tests`
- `docs_to_update`
- `handoff_artifacts`
