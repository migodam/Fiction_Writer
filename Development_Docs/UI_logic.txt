我会按真实软件的结构来写，类似：

```
VSCode / Premiere / Unreal
```

的 IDE 逻辑。

结构分为：

1️⃣ 全局 UI 结构
2️⃣ Activity Bar（左侧竖栏）
3️⃣ Sidebar（第二层导航）
4️⃣ Workspace 页面
5️⃣ Inspector Panel
6️⃣ Timeline 交互逻辑
7️⃣ Graph 交互逻辑
8️⃣ Writing Studio
9️⃣ Simulation
10️⃣ Consistency
11️⃣ Beta Reader
12️⃣ Publishing
13️⃣ Project Insights

---
